# Org Overview is empty until this repo is public

**GitHub shows the org’s main page only from `profile/README.md` in a *public* `.github` repo.**

If the org Overview tab is empty:

1. Open **Settings** (this repo).
2. **General** → scroll to **Danger Zone**.
3. **Change repository visibility** → choose **Public** → confirm.

After that, the Overview will show the content of `profile/README.md`.

See [docs/org-profile-readme.md](docs/org-profile-readme.md) for details.
